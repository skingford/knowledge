## 推荐书单

### 入门到进阶

- `Effective Go`
  适合：入门到中级
  作用：建立 Go 代码风格和常用写法认知

- `Go 语言实战` 或同类入门书
  适合：初学者
  作用：补齐语言和基础工程实践

### 并发与底层

- `Go Concurrency Patterns`
  适合：中级
  作用：理解 Go 并发模型的经典写法和设计思路

- `Go Memory Model`
  适合：中高级
  作用：理解并发可见性、同步边界和正确性问题

### 工程与系统设计

- `100 Go Mistakes and How to Avoid Them`
  适合：中高级
  作用：从常见错误反推工程最佳实践

- `Cloud Native Go` 或同类服务端工程书
  适合：中高级
  作用：把 Go 放进微服务、云原生和工程场景里理解

## 推荐博客与官方资料

### 官方资料

- `Go 官方文档`
  重点看：语言规范、标准库文档、Memory Model、Effective Go

- `Go Blog`
  重点看：并发、错误处理、性能优化、调度、GC

### 源码阅读

- `runtime`
  重点看：调度、内存分配、GC

- `sync`
  重点看：Mutex、RWMutex、WaitGroup、Pool、Map

- `net/http`
  重点看：客户端连接复用、服务端请求处理、中间件模型

### 中文资料

- 极客时间 Go 专栏
- 掘金和高质量技术博客
- GitHub 上成熟 Go 服务项目的代码组织方式

## 推荐视频与课程

- Go 官方或社区分享视频
- GopherCon 相关演讲
- YouTube 上的 Go 并发、Runtime、Performance 主题分享
- 国内高质量 Go 课程与源码分析视频

重点方向：

- 并发模型
- GMP 调度
- GC 与逃逸分析
- Pprof 与性能优化
- 微服务与工程实践

## 推荐阅读顺序

1. 先看 `Effective Go`
2. 再补语言和标准库基础
3. 再看并发模型和 `Go Concurrency Patterns`
4. 再学 `Go Memory Model`、GC、逃逸分析
5. 再结合 `runtime`、`sync`、`net/http` 源码深入
6. 最后结合工程项目和性能排障专题反复复盘

## 推荐使用方式

- 先按当前面试薄弱项选资料，不要一次全看
- 看完一个主题后，补一份自己的总结和示例代码
- 每读完一类资料，最好回到题库和回答模板再输出一遍
